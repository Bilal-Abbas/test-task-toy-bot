# Test Task Toy Bot

## Dependencies
* Ruby 3.0.0

## Setup
```
git clone https://github.com/prodis/toy_robot.git
cd test-task-toy-bot
bundle install
```

## Running
`ruby main.rb [commands.txt]`

Inside `data` directory there are some test case files.

Example:
```
$ ruby main.rb data/test_case1.txt
Output: 3,3,NORTH
```

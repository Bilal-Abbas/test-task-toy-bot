# Test Task Toy Bot

## Dependencies
* Ruby 3.0.0

## Setup
```
git clone https://github.com/Bilal-Abbas/test-task-toy-bot.git
cd test-task-toy-bot
bundle install
```

## Running
`ruby main.rb data/file.txt`

Inside `data` directory there are some test case files.

Example:
```
$ ruby main.rb data/test_case1.txt
Output: 0,1,NORTH
```
